### Week 5 Objectives:

- Understand data storage services on AWS.
- Practice using Amazon DynamoDB, Amazon RDS, and AWS Secrets Manager.
- Understand how AWS Lambda accesses databases securely.

### Tasks to be carried out this week:

| Day | Task                                                                                                                                    | Start Date | Completion Date | Reference Material                         |
| --- | --------------------------------------------------------------------------------------------------------------------------------------- | ---------- | --------------- | ------------------------------------------ |
| 2   | - Learn about Amazon DynamoDB.<br>- Partition Key and Sort Key.<br>- Practice creating data tables.                                     | 18/05/2026 | 18/05/2026      | https://docs.aws.amazon.com/dynamodb       |
| 3   | - Practice reading, writing, updating, and deleting data on DynamoDB using AWS SDK for JavaScript v3.                                   | 19/05/2026 | 19/05/2026      | AWS SDK Documentation                      |
| 4   | - Learn about Amazon RDS and RDS Proxy.<br>- Compare RDS and DynamoDB.<br>- Understand how to optimize database connections.            | 20/05/2026 | 20/05/2026      | https://docs.aws.amazon.com/rds            |
| 5   | - Learn about AWS Secrets Manager.<br>- Store and manage database connection information.<br>- Practice retrieving Secrets from Lambda. | 21/05/2026 | 21/05/2026      | https://docs.aws.amazon.com/secretsmanager |
| 6   | - Build Lambda functions connecting to DynamoDB and RDS.<br>- Test data retrieval capabilities and error handling.                      | 22/05/2026 | 22/05/2026      | AWS Documentation                          |

### Week 5 Achievements:

- Understand the differences between DynamoDB and Amazon RDS.

- Successfully perform CRUD operations on DynamoDB.

- Understand the role of RDS Proxy in optimizing database connections.

- Know how to use AWS Secrets Manager to manage sensitive information.

- Successfully connect AWS Lambda with DynamoDB and RDS.

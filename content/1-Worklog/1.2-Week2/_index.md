### Week 2 Objectives:

- Understand AWS services for deploying frontend interfaces and user authentication.
- Practice deploying static websites on Amazon S3.
- Understand the mechanisms of CloudFront, WAF, and Amazon Cognito.

### Tasks to be carried out this week:

| Day | Task                                                                                                        | Start Date | Completion Date | Reference Material                     |
| --- | ----------------------------------------------------------------------------------------------------------- | ---------- | --------------- | -------------------------------------- |

| 3   | - Practice creating S3 Bucket.<br>- Upload static website.<br>- Configure Static Website Hosting.           | 28/04/2026 | 28/04/2026      | AWS Documentation                      |

### Week 2 Achievements:

- Understand the role of Amazon S3 in data storage and static website deployment.

- Successfully create and configure S3 Bucket.

- Know how to use CloudFront for content distribution.

- Understand the functionality of AWS WAF in application protection.

- Get familiar with Amazon Cognito and user authentication process.

- Understand the relationships between S3, CloudFront, and Cognito in system architecture.
